* Tools Used
** Python
We used the following libraries: =numpy=, =matplotlib.pyploy=, =statsmodels=, =scipy=, =pandas=, =seaborn=.
** Orgmode and Ein
We followed a style of Literate Programming, proposed by Donald
Knuth. So, we mixed code with the text-solution in one file:
=Solution.org=.

This is possible because we are using Org-mode in Emacs and
Ein-python -  a port from Jupyter to Emacs - to avaluate python code.
